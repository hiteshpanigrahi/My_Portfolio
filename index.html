Latest Projects

PID Line Follower

Built using Arduino Nano, TB6612FNG motor driver, and a 7-channel IR array, this high-speed bot precisely tracks black lines. Powered by a 7.2V LiPo battery and 600 RPM N20 gear motors. All the components are soldered on a perfboard for a neat design.

GitHub

16-DOF Humanoid Bot

Powered by Arduino Mega and 16 MG995 servo, this Bluetooth-controlled robot (via HC-05) features a hand-crafted sheet metal body. It performs walking, dancing, and various motions, running on dual 3.7V lithium-ion cells.

GitHub

8-DOF Humanoid Bot

Powered by ESP32 and 8 MG995 servos, this robot has a OLED for facial expressions and voice capabilities. Controlled via a custom-designed WiFi app with joystick controls and preset movements, it can walk, talk, and dance.

GitHub

IoT Based Smart LED Matrix

A 32x8 LED matrix powered by ESP8266 and MAX7219 drivers. Displays an NTP clock, weather, temperature, humidity, news, and custom messages. Configurable via a web interface, with data fetched using API integration. Built with 5mm plywood.

GitHub

Hituino v1

A barebones Arduino designed from scratch using KiCAD. Compact and efficient, it offers full GPIO, analog, and digital pin functionality like the Uno R3. Ideal for DIY projects, it’s cost-effective, optimized, and tailored for versatile applications.

GitHub

IoT Based Home Automation

An IoT-based home automation system using the ESP12E module to control up to 4 appliances remotely from anywhere in the world. It offers seamless control and real-time status updates. The custom PCB is designed using KiCAD for compactness.
